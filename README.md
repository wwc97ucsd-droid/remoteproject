# 2048

基于单文件 HTML 的经典 **2048** 小游戏，界面与说明为中文，无需构建或安装依赖。

## 功能

- 4×4 棋盘，合并相同数字直至出现 **2048** 即胜利
- **分数** 与 **最佳分数**（最佳分数保存在浏览器 `localStorage`，键名 `best2048`）
- 键盘 **方向键** 操作；在触摸屏上可在棋盘上 **滑动** 移动（滑动阈值约 30px）
- **重新开始** / 游戏结束后的 **再来一局**
- 合并与新生成方块带有简单动画；窄屏下布局自适应

## 如何运行

在浏览器中直接打开项目根目录下的 `index.html` 即可。

若因本地文件策略导致脚本受限，可用任意静态文件服务打开该目录，例如：

```bash
npx --yes serve .
```

然后在终端提示的地址中打开页面。

## 技术说明

- 纯 **HTML + CSS + JavaScript**，无第三方库
- 逻辑与样式均写在 `index.html` 中
